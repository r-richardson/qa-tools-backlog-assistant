# QE Tools Backlog Status

This is the dashboard for [QE Tools](https://progress.opensuse.org/projects/qa/wiki/Tools).

Implemented using [openSUSE/backlogger](https://github.com/openSUSE/backlogger) (custom fork).

## Features
-   **Modern Dashboard:** Responsive design with openSUSE branding.
-   **App Links:** Configurable list of tool links with automatic favicon fetching.
-   **Smart Grouping:** Failing queries are highlighted at the top.
-   **Custom Icons:** Place overrides in the `icons/` directory.

See https://os-autoinst.github.io/qa-tools-backlog-assistant/
